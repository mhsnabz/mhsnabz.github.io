VayaApp

This application can only be used by university students to register with their @edu.tr e-mail address.
With this application, students and lecturers can make announcements about courses.
Users can message, make announcements to school groups and receive instant notifications.



This app avaible on Google PlayStore and AppStore This application was written with native android and native swift. Technologies I use in this application * Java * Swift 5 * Firebase / Firestore * JavaScript (for cloud messaging)
